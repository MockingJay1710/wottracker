## WOTTracker

Application Windows développée pour mesurer l’activité et les heures supplémentaires des employés.  
Elle récupère automatiquement les données via **Windows Event Log**, garantissant zéro perte d’informations, et propose un **tableau de bord en temps réel** avec génération automatique de rapports.

**Technologies :** C#, .NET, WinForms, SQL Server Compact, Serilog
